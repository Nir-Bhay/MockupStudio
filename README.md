# Mockup Studio Pro 🚀

**Mockup Studio Pro** is a high-performance, browser-based design tool crafted for developers and designers to create stunning, professional device mockups in seconds. Turn your raw screenshots into export-ready marketing visuals for X, LinkedIn, and portfolios.

## ✨ Key Features

- **Multi-Device Layouts:** Choose from 4 powerful presets—Hero (Desktop), Duo (Desktop + Phone), Trio (All Devices), and Stack.
- **Dynamic Backgrounds:** 12+ premium gradients (Sahara, Midnight, Aurora, etc.) or set your own custom color.
- **Pixel-Perfect Customization:** 
  - Adjust device frame colors (White, Space Gray, Gold, Silver).
  - Control browser roundness and shadow intensity.
  - Toggle navigation bars, phone notches, and reflection effects.
- **High-Resolution Export:** Download your mockups in 2x, 3x (High Quality), or 4x (Ultra) resolution in PNG or JPEG format.
- **Interactive Stage:** Zoom in/out, fit-to-view, and drag-and-drop support for a seamless workflow.

---

## 🛠 Tech Stack

- **Frontend:** HTML5, CSS3 (Modern Variables & Flexbox/Grid)
- **Engine:** [html2canvas](https://html2canvas.hertzen.com/) for high-fidelity client-side rendering.
- **Typography:** Inter (System-ui) and Playfair Display for a premium editorial feel.
- **Icons:** Custom SVG components and Emoji-based interface.

## 🚀 Getting Started

### Prerequisites
A modern web browser (Chrome, Safari, Firefox, or Edge).

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Nir-Bhay/MockupStudio.git
   ```
2. Open `index.html` in your browser. No build step required—it's zero-config!

---

## 📖 Usage Guide

1. **Upload Assets:** Click the upload zones in the sidebar to add your Desktop, Mobile, or Tablet screenshots.
2. **Choose Layout:** Select a preset that fits your project.
3. **Customize:** Tweak the backgrounds, shadows, and frame colors to match your brand.
4. **Export:** Hit "Download Mockup" to save your high-res graphic.

---

## 🎨 Design Philosophy

Mockup Studio Pro is built on the principle of **"Elegant Utility."** Every UI element is designed to stay out of your way while providing deep customization options. The "Sahara" theme (default) is inspired by premium editorial design, ensuring your mockups look like they were made by a pro agency.
